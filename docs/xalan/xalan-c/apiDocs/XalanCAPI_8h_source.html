00083     /**
00084      * Terminate Xalan and Xerces.
00085      *
00086      * Should be called only once per process after deleting all
00087      * instances of XalanTransformer.
00088      *
00089      * Once a process has called this function, it cannot use the
00090      * API until another call to XalanInitialize has been made.
00091      * 
00092      * Optionally, if the ICU has been integrated, this will
00093      * call the ICU clean up function.  This must only be done
00094      * if the ICU will no longer be used by the process, since
00095      * the ICU will no longer be in a usable state.  See the
00096      * ICU documentation for more details.
00097      *
00098      * This is handy when using leak-detection software, as all
00099      * static data allocated by Xalan (and optionally, the ICU)
00100      * will be freed.
00101      *
00102      * @param fCleanUpICU If true, call the ICU clean up function.
00103      */
